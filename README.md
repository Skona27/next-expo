# next-expo
Monorepo template for app with Nextjs as a web framework and Expo for Android and iOS.

This repository is made to provide an easy way of starting new, cross platform projects. 


## Getting started

```bash
cd next-expo
```

```bash
yarn
```

### Development

web:
```bash
cd packages/web
yarn dev
```

native:
```bash
cd packages/native
yarn start
```
